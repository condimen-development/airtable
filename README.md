# About Airtable

Airtable is a low-code database platform to create custom built applications. This repository stores the scripts that condimen has developed for it.

## Assign numbers
This script contains several functions that help to create random, non-sequential register numbers.

### `getRest()`
Get a new array with the characters in the second argument removed from the first argument


### `combineUnique()`
Create a combination without repeating characters from argument array and return it as permutation array

### `getRandom()`
Get a random number from generated set
